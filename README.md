# Who am I in the Periodic Table?

This is an interactive game inspired by the classic "Who am I?", developed as a pedagogical tool for Chemistry education. The project was born from the need to make learning the Periodic Table more dynamic, inclusive, and accessible to all students.

---

### 🕹️ About the Game
**"Who am I in the Periodic Table?"** is a pedagogical adaptation transposed into the universe of chemistry, functioning as a guessing tool where players must identify chemical elements through strategic clues. Each element is presented through a unique "identity card" that harmonizes rigorous technical data — such as atomic number and mass — with curiosities and practical everyday applications, making learning more tangible.

The dynamics were structured under the pillars of active methodology, prioritizing deductive reasoning over simple mechanical memorization. In this way, students are encouraged to understand the organizational logic of the table and periodic properties while playing. To support this experience, the digital platform acts as an interactive arm, offering a quick consultation system with real-time visual and auditory support, ensuring that the flow of the game is fluid and informative for all participants.

**The Accessibility Differentiator:**
The main pillar of this project is **Educational Equity**. We have developed a solution that bridges the digital and the physical:
1.  **Web Accessibility:** Implementation of automatic voice narration that describes the element and its application, allowing interaction for students with visual impairments.
2.  **Tactile Material (Braille):** Production of physical cards adapted with **Braille writing**, ensuring that blind students have access to the same information as their peers, promoting real inclusion in the classroom.
